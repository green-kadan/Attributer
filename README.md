# Attributer
- Attributer は、タイムスタンプ、ドキュメントプロパティ、属性を変更するだけでなく、ゾーン情報を削除できるWindowsのエクスプローラの拡張シェルプログラムです。
- Attributer はWata氏が作成したものですが、公開を止められて不具合もあったので私が修理したものを公開したものです。
